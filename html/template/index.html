媽媽：俞曉琳a

10/2017

教學到底是什麼？當我們說我們在「教」別人東西時是什麼意思？教學時主要面臨的問題是什麼？我們應該用什麼教學方法？為什麼用這種方法？什麼是有助於學生學習的方法？又是什麼會讓學生不想學習？以下幾個案例可以幫助教師認識對教學的不同理解方式，並且可以鼓勵授課者從不同的觀點進行自我反思，從而了解自己的問題。

學生優勢

  • 可以按照我的步調學習並養成自學的好習慣
  • 讓我一次就學會學寫程式而且英文也進步了
  • Coding for Fun教材讓我自信快樂學習程式

家長優勢

  • 讓我的孩子能按照自己的節奏學習，提高學習興趣並且增加自信心
  • 透過探索和主動思考的非填鴨式教學，讓孩子擁有解決問題的能力
  • Coding for Fun培養孩子帶得走的程式自學力又能提高學英文動機

教師優勢

  • 提供教學技巧與專業素養之認證培訓，滿足程式語言設計教學的師資需求
  • 老師引導和鼓勵孩子勇敢嘗試，配合Coding for Fun教材增強程式自學力
  • Coding for Fun完整的基礎應用與進階的師資培訓是老師教學的最佳後盾

基礎班

  • 唯一使用翻轉教室教學
  • 入手快，短時間就能學會並應用
  • 運用動畫和遊戲提高學習意願與樂趣

密集班

  • 唯一使用翻轉教室教學模式
  • 最實用，語法簡單而且易學易懂
  • 美國頂尖大學常用的入門程式語言

工作坊

  • 讓學生擁有實用的技能，成為國際頂尖人才
  • 根據學生個人特質及興趣，建議適合的課程
  • 程式自主學習，老師從旁輔助學生克服英文障礙
