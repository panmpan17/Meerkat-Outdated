在我們成長的過程中，對「教學」的認知，就是老師站在前面講課。
老師也許會詢問大家「聽懂了嗎？」「有沒有問題？」當老師認為同學跟上進度時，老師會繼續往下授課。

但是

  1. 老師允許學生提問嗎？
  2. 學生會不會因為同儕壓力而裝懂？
  3. 老師何時繼續往下教？所有的學生都懂了？八成的學生懂了？還是大家不出聲，就繼續往下教？

不同的課程，不同的老師，我們會得到不同的答案。

到底要如何教，可以讓孩子學得更好，學得快樂有自信呢？

首先，我們必須承認每個人的學習方式是不同的，也有各自學習的步調。

比方說，有些人理解得快，忘得也快；有些人理解得慢，但懂了就永遠忘不了。如何塑造一個環境，孩子可以自在的學習、有效的學習呢？

翻轉教室是一種不錯的方式。

樂學科技的翻轉教室結合程式設計學習的特性，已有相當好的教學成效。

首先，我們依據孩子思考的脈絡與學習心裡，設計了程式基礎課程教學課程。

在課堂上，老師引導孩子每看完一段影片，隨即動手嘗試與體驗。
若有看不懂的部分，孩子可以反覆觀看影片。
即使分心了，很快地又可以回到課程中。

老師的角色，並不是催趕孩子的進度，而是隨時救援，為孩子講解或排除狀況。
聰明的孩子則一路往前學習，完全不需要等待老師下一個指令，順暢地完成學習。

在課堂上，每個孩子都沈浸在課程中。看似平淡、不熱絡，但每個孩子的學習都是滿滿的收穫。

我們的 Scratch 基礎課程與Python基礎課程都是這樣的模式，為孩子建立最扎實的程式能力根基。孩子不僅學得好，學得有自信，更學到自學的能力！


美國勞動部在2015年的研究報告〈未來工作—21世紀工作的趨勢與挑戰〉中預測，孩子未來從事的工作，有65％現在還不存在。

目前是資訊爆炸的時代，在網路搜尋就能取得知識與訊息。傳統背誦、記憶、反覆練習的學習方式, 能幫助孩子面對未來嗎？

傳統的學習方式並非全然無用, 但孩子還需要「學習的能力」。也就是「自學力」。

自學力包括

  1. 嘗試的能力
  2. 思考、思辨的能力
  3. 找尋答案、解決問題的能力
  4. 閱讀能力，包括英文能力

樂學科技的程式課程, 啟發學生的學習動機, 發展自信心，讓孩子養成自學力。

有別於一般課程, 樂學以翻轉教室啟動孩子的自學力, 讓孩子擺脫被動的學習心態, 逐漸轉為主動探索與思考。
我們不以標準答案限制孩子的思維, 開啟孩子的創意與天賦。

這個帶得走的自學力不僅適用於程式學習, 也可以應用於各領域的學習。
