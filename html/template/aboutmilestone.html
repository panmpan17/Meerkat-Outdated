2012年 7月

創辦人投入兒童程式教材的研究與發展。

2013年寒假

第一次進行實驗教學，發現孩子學程式的秘訣。

2014年 4月

開始經營部落格 kidscoding.tw，與 Facebook 粉絲團「兒童程式設計 KidsCoding」，推廣兒童程式設計。

2014年 7月

推出第一代 Python 初級課程，以翻轉教室教授 Python 基礎課程。

2014年 10月

以翻轉教室教授 Scratch 基礎課程，並推出公益線上免費課程 Coding for Fun。

2014年 11月 ~ 2015年 1月

FlyingV 群眾募資「Coding for Fun 快樂寫程式」沒有成功。

2015年 3~6月

推出公益實體課程 Coding for Fun 程式工作坊。

2015年 6月

旺宏電子演講：「看見未來新趨勢 — 程式設計師的世界」。

2016年 6月

新北市國中小資訊組長與教師研習演講：「如何教授運算思維與程式語言教學」。

2016年 6-11月

Coding for Fun 公益師資培訓，台北、新竹、台中、台南、高雄共舉辦 10 場。培訓師資 100 多人。

2016年 7月

推出第二代 Python 初級課程。

2016年 7月

由三采文化出版圖書「讓孩子快樂學程式」。

2016年 7~9月

勾勾 GoGo 群眾募資「Coding for Fun 快樂學程式偏鄉推廣計畫」，將 Coding for Fun 教材與「讓孩子快樂學程式」一書寄至偏鄉國中小，共 900 多所。

2016年 9月

教育廣播電台「教育好夥伴：父母玩很大」常玉慧專訪「讓孩子快樂學程式」。

2016年 9月 ~ 2017年 2月

與富志科技合作，向財團法人種子教育基金會提案「新竹地區 Coding for Fun 兒童程式推廣計畫」，提供新竹地區 3 所學校免費 Scratch 社團課程與師資培訓 60 人。

2016年 10月

中原大學「電機講座」專題演講：「看見未來新趨勢 — 軟體吃掉全世界」。

2016年 11月

屏東縣國中小資訊組長與教師研習演講「看見未來新趨勢 — 兒童學程式」。

2016年 11月

嘉義市志航國小教師研習演講「看見未來新趨勢 — 兒童學程式」。

2017年 1月

成立樂學科技股份有限公司，以公益為初心，推廣程式設計教材。

2017年 3月

屏東縣屏東女中學生與教師各一場演講「看見未來新趨勢 — 輕鬆踏入程式設計的世界」。

2017年 3~5月

新北市大埔國小教師研習演講「看見未來新趨勢 — 兒童學程式」。

2017年 4~6月

授權培訓並認證第一批「Coding for Fun 樂學科技授權老師」。

2017年 6月

中鼎工程演講。講題：「看見未來新趨勢 — 軟體吃掉全世界」。

2017年 5月 ~ 2019年5月

無償授權「Coding for Fun 兒童程式設計推廣教材」予財團法人種子教育基金會做非營利推廣。

2017年 8月

台東區網中心演講。講題：「兒童程式設計教學研習─讓孩子快樂學程式」。

2017年 8月

環宇廣播電台專訪「交大幫幫忙:用程式軟體帶動創新創意」。

2017年 9月

2017 新北市立圖書館三峽北大分館 「親親我的寶貝」親子科技教育講座「讓孩子快樂學程式」。

2017年 9月

中央大學學習與教學研究所 Seminar 客座 2 天共 4 小時課程，以「Coding for Fun」學 Scratch。

2017年 10月

台灣婦女團體全國聯合會舉辦數位時代‧台灣女子風光座談會「讓孩子快樂學程式」。

2017年 11月

行天宮附設玄空圖書館「圖像化遊戲程式設計體驗秀」，帶領銀髮族學 Scratch。

2017年 11月 ~ 2018年 3月

授權培訓並認證第二批「Coding for Fun 樂學科技」授權老師。

2018年 3月

中央大學師資培育課程與學習與教育研究所客座講座「彈指之間 創見未來」。

2018年 3月

中廣新聞網氣象達人彭啟明專訪「從小開始學寫程式」。

2018年 4~6月

授權培訓並認證第三批「Coding for Fun 樂學科技」授權老師。
